# bmsg
bmsg (b message) is a simple message encryption and decryption tool using RSA and a Tkinter GUI.

## Usage
Install:
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
Run:
```bash
python main.py
```
